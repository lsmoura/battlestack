# Battle Stack - CAD

This work is a recreation of the battlestack 3d print project created by 
Twogz. The original battlestack buid can be found for free on 
[printables](https://www.printables.com/model/1332602-battlestack)
and other websites.

The motivation of creating this is that I wanted to use parameters to be
able to change the file to have the magnets being inserted after the 
print, rather than during the print job, and the recreation of the 
original objects was a part of the journey.
